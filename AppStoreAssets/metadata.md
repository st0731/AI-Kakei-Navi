## 説明文（4000文字以内）

```
📸 撮るだけで、支出を自動記録。プライバシー完全守護のAI家計管理アプリ。

「AI家計ナビ」はレシートを撮影するだけで、品目・金額・店名を自動で読み取り、支出を記録・分析するアプリです。すべての解析をiPhone上で完結するため、撮影したレシート画像や支出データが外部サーバーに送信されることは一切ありません。


【主な機能】

▶ レシート自動読み取り（OCR）
カメラで撮影するか、写真ライブラリから選択するだけ。品目・金額・店名・日付を自動で認識し、手入力の手間をゼロにします。

▶ オンデバイスAI解析
すべての解析はあなたのiPhone上で動くAI（Qwen3-1.7B）が実行します。クラウドサービスやAPIへのデータ送信はありません。インターネット接続なしで全機能を使えます。

▶ 節約AIアドバイス（チャット形式）
過去の支出データをもとに、AIが具体的な節約アドバイスをチャット形式で提供します。「今月の食費が多い理由は？」「節約できる項目は？」など、自然な日本語で相談できます。

▶ グラフ・支出分析
月別・カテゴリ別のグラフで支出パターンを一目で把握。「必要」「便利」「贅沢」の3段階スコアで、支出の優先度管理もできます。

▶ 手動編集・履歴管理
自動読み取り結果を手動で修正できます。過去のレシート一覧から編集・削除も自由自在。

▶ 完全オフライン対応
初回のAIモデルダウンロード（約1GB・Wi-Fi推奨）が完了すれば、以降はインターネット接続なしで全機能を利用できます。


【プライバシーへのこだわり】

✓ すべての処理はデバイス内で完結
✓ レシート画像・支出データは外部へ一切送信しない
✓ 広告表示なし・トラッキングなし
✓ クラウドサービスへのデータ連携なし
✓ サブスクリプションなし・APIコストなし


【こんな方におすすめ】

・レシートの手入力が面倒な方
・クラウドへのデータ送信に抵抗がある方
・プライバシーを大切にしながら家計管理したい方
・AIを生活に活用してみたい方
・節約や支出の見直しをしたい方


【ご利用にあたって】

AI機能をお使いいただくには、初回起動時に約1GBのAIモデルデータのダウンロードが必要です。Wi-Fi環境でのダウンロードを推奨します。一度ダウンロードすれば、以降はオフラインで動作します。
```

## App Review 審査官へのメモ（Review Notes）

```
【デモアカウント】
このアプリにはログイン機能がないため、デモアカウントは不要です。

【AI機能について】
このアプリはオンデバイスLLM（Qwen3-1.7B-4bit、約1GB）を使用しています。
すべての推論処理はデバイス上で実行され、ユーザーデータが外部サーバーに送信されることはありません。

モデルのダウンロード元: https://huggingface.co/mlx-community/Qwen3-1.7B-4bit

【レシートOCRのテスト方法】
レシートが手元にない場合は、以下の画像をご利用いただけます:
　・スーパーやコンビニのレシートの写真
　・またはアプリ内の「手動入力」から直接データを入力できます

【ネットワーク通信】
以下の場合のみネットワーク通信が発生します:
　1. AIモデルの初回ダウンロード時（huggingface.co）
　2. 上記以外の通信は行いません

【カメラ・写真ライブラリ】
レシートを撮影・選択するためのみ使用します。
撮影した画像は外部に送信されず、OCR処理後に破棄されます（保存はしません）。
```
